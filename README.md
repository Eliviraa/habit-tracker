# Трекер привычек

Настольное приложение для отслеживания ежедневных привычек.

## О проекте

Приложение помогает пользователю:
- создавать список привычек
- отмечать выполнение привычек каждый день
- отслеживать серии выполнения (сколько дней подряд)
- смотреть график прогресса

## Технологии

- C++17
- Qt 6 (виджеты)
- SQLite
- CMake

## Требования для запуска

- Windows 10/11
- Qt 6 (устанавливается отдельно)

## Установка и запуск

1. Установить Qt 6 с официального сайта
2. Открыть проект в Qt Creator
3. Нажать "Собрать" и "Запустить"
